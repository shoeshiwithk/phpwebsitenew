Functions


In PHP, a function is a reusable block of code that performs a specific task. Functions can take parameters and return values. Here’s how to define and use functions in PHP.

Function Definition

The basic syntax for defining a function is as follows:
function functionName($parameter1, $parameter2) {
    // Code to execute
    return $result; // Optional
}

Example
Here’s a simple example of a function that adds two numbers:
function add($a, $b) {
    $sum = $a + $b;
    return $sum; // Return the result
}

// Calling the function
$result = add(5, 10);
echo "The sum is: " . $result; // Output: The sum is: 15

Breakdown
    1.Function Declaration: The function is declared using the function keyword, followed by the function name and parentheses for parameters.
    2.Parameters: In the example, $a and $b are parameters that the function accepts.
    3.Function Body: The code inside the curly braces is executed when the function is called.
    4.Return Statement: The return statement sends the result back to the caller. If a function does not return a value, it will return NULL by default.

Formal Parameter vs Actual Parameter

Formal Parameters Actual Parameters
Variables listed in functions declaration. Values or variables passed to function when called.
Serves as placeholders for the values that will be used in the function. Provide the actual data that will be processed by the function.
Limited to the function where they are defined. Can be variables from outside the function or literal values.
In function add($a, $b), $a and $b are formal parameters. In add(5, 10), 5 and 10 are actual parameters.
Can be typed (e.g., type hinting in PHP 7 and later). Can be of any type (e.g., integer, string, array).
Used within the function body to perform operations. Passed during the function call to invoke the function.

Recursion

Recursion in PHP refers to the process where a function calls itself in order to solve a problem. It is often used to break down complex problems into simpler sub-problems. Here’s a breakdown of how recursion works, along with an example.

Key Components of Recursion
Base Case: The condition under which the recursion stops. This prevents infinite loops and eventually leads to a return value.
Recursive Case: The part of the function where the function calls itself with modified arguments.

Example: Factorial Function
A common example of recursion is calculating the factorial of a number, which is the product of all positive integers up to that number.
<?php
function factorial($n) {
    // Base case
    if ($n <= 1) {
         return 1;
    }
    // Recursive case
    return $n * factorial($n - 1);
}

// Example usage
echo factorial(5); // Output: 120
?>

String Handling

Interpolation with Curly Braces

In PHP, interpolation with curly braces is a way to embed variables within double-quoted strings or heredoc strings. This approach allows you to clearly define variable boundaries, especially when dealing with complex expressions or when concatenating strings.

Basic Interpolation
When you use double quotes, PHP automatically interpolates variables:
$name = "John";
echo "Hello, $name!"; // Output: Hello, John!

Using Curly Braces for Clarity
You can use curly braces to clearly define the variable, which is particularly useful in the following scenarios:

1.When the variable is followed by text: Curly braces help distinguish where the variable name ends.
$name = "John";
echo "Hello, {$name}Doe!"; // Output: Hello, JohnDoe!

2.For array elements: When accessing array elements, curly braces can make it clearer.
$arr = ['a' => 'Apple', 'b' => 'Banana'];
echo "The first letter is {$arr['a']}."; // Output: The first letter is Apple.

3.For complex expressions: When using complex expressions, curly braces help ensure that PHP parses the expression correctly.
$a = 5;
$b = 10;
echo "The sum of a and b is {$a + $b}."; // Output: The sum of a and b is 15.

String Operators

In PHP, string operators are used to manipulate and concatenate strings. Here are the primary string operators you'll encounter:

1. Concatenation Operator (.)
The dot operator (.) is used to concatenate two or more strings.

2. Concatenation Assignment Operator (.=)
This operator appends the right string to the left string and assigns the result to the left string.

3. String Length Operator (strlen())
While not an operator per se, the strlen() function is commonly used to get the length of a string.

4. String Comparison Operators
These operators compare strings, typically using the following:

==: Checks if two strings are equal.
===: Checks if two strings are equal and of the same type.
!=: Checks if two strings are not equal.
!==: Checks if two strings are not equal or of different types.
<: Compares strings based on ASCII value (lexicographical order).
>: Similar to <, but checks if the first string is greater than the second.
<= and >=: Check for less than or equal to, and greater than or equal to, respectively.

String Functions

PHP offers a variety of built-in string functions that make it easy to manipulate and work with strings. Here’s a list of some commonly used string functions along with brief descriptions and examples.

1. strlen()
Returns the length of a string.
$text = "Hello, World!";
echo strlen($text); // Output: 13

2. strtoupper()
Converts a string to uppercase.
$text = "Hello";
echo strtoupper($text); // Output: HELLO

3. strtolower()
Converts a string to lowercase.
$text = "HELLO";
echo strtolower($text); // Output: hello

4. trim()
Removes whitespace (or other specified characters) from the beginning and end of a string.
$text = " Hello, World! ";
echo trim($text); // Output: "Hello, World!"

5. substr()
Returns a portion of a string.
$text = "Hello, World!";
echo substr($text, 7, 5); // Output: World

6. strpos()
Finds the position of the first occurrence of a substring in a string.
$text = "Hello, World!";
echo strpos($text, "World"); // Output: 7

7. str_replace()
Replaces all occurrences of a search string with a replacement string.
$text = "Hello, World!";
echo str_replace("World", "PHP", $text); // Output: Hello, PHP!

8. explode()
Splits a string by a string delimiter and returns an array.
$text = "apple,banana,orange";
$fruits = explode(",", $text);
print_r($fruits); // Output: Array ( [0] => apple [1] => banana [2] => orange )

9. implode()
Joins array elements into a string using a specified delimiter.
$fruits = ["apple", "banana", "orange"];
echo implode(", ", $fruits); // Output: apple, banana, orange

10. strpos()
Finds the position of the first occurrence of a substring in a string.
$text = "Hello, World!";
echo strpos($text, "World"); // Output: 7

Arrays

PHP Arrays

In PHP, arrays are a versatile data structure that allows you to store multiple values in a single variable. PHP supports both indexed (numeric) arrays and associative arrays, which use named keys. Here’s an overview of PHP arrays, including their types, creation, and common functions.

Types of Arrays

1.Indexed Arrays
 Arrays where the index is a number.
Created using the array() function or the short array syntax [].

Example:
$fruits = array("Apple", "Banana", "Cherry");
// or
$fruits = ["Apple", "Banana", "Cherry"];

echo $fruits[0]; // Output: Apple

2.Associative Arrays
 Arrays where the index is a string (key).
Useful for storing data in key-value pairs.

Example:
$person = [
"first_name" => "John",
"last_name" => "Doe",
"age" => 30
];

echo $person["first_name"]; // Output: John

3.Multidimensional Arrays
 Arrays that contain one or more arrays.
Useful for representing complex data structures.

Example:
$contacts = [
"John" => ["email" => "john@example.com", "phone" => "123-456-7890"],
"Jane" => ["email" => "jane@example.com", "phone" => "987-654-3210"]
];

echo $contacts["John"]["email"]; // Output: john@example.com

Accessing array elements

You can access elements by their index (for indexed arrays) or key (for associative arrays).

Example:
echo $fruits[1]; // Output: Banana
echo $person["age"]; // Output: 30

Deleting from array

In PHP, you can delete elements from arrays using several functions and methods. Here's how to remove elements from both indexed and associative arrays.

1. Using unset()

The unset() function removes a specific element from an array by its index or key.

Example with Indexed Arrays:
$fruits = ["Apple", "Banana", "Cherry"];
unset($fruits[1]); // Removes "Banana"
print_r($fruits); // Output: Array ( [0] => Apple [2] => Cherry )

Example with Associative Arrays:
$person = [
"first_name" => "John",
"last_name" => "Doe",
"age" => 30
];
unset($person["age"]); // Removes the "age" key
print_r($person); // Output: Array ( [first_name] => John [last_name] => Doe )

2. Using array_splice()

The array_splice() function removes a portion of the array and can also replace it with new elements.

Example:
$fruits = ["Apple", "Banana", "Cherry", "Date"];
array_splice($fruits, 1, 2); // Removes 2 elements starting from index 1
print_r($fruits); // Output: Array ( [0] => Apple [1] => Date )

3. Using array_pop()

The array_pop() function removes the last element from an array.

Example: $fruits = ["Apple", "Banana", "Cherry"];
$lastFruit = array_pop($fruits); // Removes "Cherry"
print_r($fruits); // Output: Array ( [0] => Apple [1] => Banana )

4. Using array_shift()

The array_shift() function removes the first element from an array.

Example:
$fruits = ["Apple", "Banana", "Cherry"];
$firstFruit = array_shift($fruits); // Removes "Apple"
print_r($fruits); // Output: Array ( [0] => Banana [1] => Cherry )

5. Using array_filter()

You can also remove elements conditionally using array_filter(). This function creates a new array with elements that pass a given test.

Example:
$numbers = [1, 2, 3, 4, 5];
$evenNumbers = array_filter($numbers, function($num) {
return $num % 2 == 0; // Keep only even numbers
});
print_r($evenNumbers); // Output: Array ( [1] => 2 [3] => 4 )

Iterating with each() and foreach()

1. each()

The each() function was used to return the current key and value pair from an array and move the internal pointer forward. However, it has been deprecated as of PHP 7.2, so it’s recommended to use foreach instead.

Syntax
list($key, $value) = each($array);

Example
$fruits = ["Apple", "Banana", "Cherry"];
reset($fruits); // Reset the array pointer

while (list($key, $value) = each($fruits)) {
echo "$key: $value\n";
}
// Output:
// 0: Apple
// 1: Banana
// 2: Cherry

2. foreach()

The foreach construct is used to iterate over arrays in a straightforward way. It is commonly used because of its simplicity and readability.

Syntax:
foreach ($array as $value) {
// Code to execute for each $value
}

You can also get both keys and values:
foreach ($array as $key => $value) {
// Code to execute with $key and $value
}

Example:
$fruits = ["apple", "banana", "cherry"];

foreach ($fruits as $fruit) {
echo $fruit . "\n";
}

$associativeArray = ["a" => "apple", "b" => "banana", "c" => "cherry"];

foreach ($associativeArray as $key => $value) {
echo "$key: $value\n";
}

Iterative functions

In PHP, the functions current(), next(), prev(), reset(), and end() are used to manipulate the internal pointer of an array. These functions allow you to traverse arrays in a more controlled manner than simply using loops. Here's a breakdown of each function:

1. current()

Returns the value of the element pointed to by the internal pointer of the array without advancing the pointer.

Example:
$array = [1, 2, 3, 4, 5];
echo current($array); // Outputs: 1

2. next()

Advances the internal pointer to the next element in the array and returns its value. If the pointer is already at the end, it returns false.

Example:
$array = [1, 2, 3, 4, 5];
next($array); // Moves pointer to the second element
echo current($array); // Outputs: 2

3. prev()

Moves the internal pointer to the previous element in the array and returns its value. If the pointer is already at the beginning, it returns false.

Example:
$array = [1, 2, 3, 4, 5];
next($array); // Move to the second element
next($array); // Move to the third element
echo prev($array); // Outputs: 2

4. reset()

Resets the internal pointer of the array to the first element and returns its value. If the array is empty, it returns false.

Example:
$array = [1, 2, 3, 4, 5];
next($array); // Move to the second element
echo reset($array); // Outputs: 1

5. end()

Advances the internal pointer to the last element of the array and returns its value. If the array is empty, it returns false.

Example:
$array = [1, 2, 3, 4, 5];
echo end($array); // Outputs: 5
